# 🧭 Path Finding Visualizer

An interactive pathfinding algorithm visualizer built using **React.js**.  
This project demonstrates how popular pathfinding algorithms work in real-time on a grid.

🔗 **Live Demo:** https://pathfinding-sable.vercel.app

---

## 📌 About the Project

This web application allows users to:

- Create walls/obstacles
- Select start and end nodes
- Visualize pathfinding algorithms step-by-step
- See how shortest paths are computed

The goal of this project is to understand graph traversal and shortest path algorithms visually.
